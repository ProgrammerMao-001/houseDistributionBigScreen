# houseDistributionBigScreen
太原市房屋分布信息大屏

```
cnpm i coordtransform
cnpm i dayjs
cnpm i echarts
```



# 数据表介绍
```
1.房屋表 housetable
	city：市名
	houseName：房屋名称
	address：地址
	salesperson：销售员	
	salespersonphone：销售员电话
	sellnull：销售员售卖楼盘数量
	position：经纬度
	houseprice：楼盘单价（元/㎡）
	totalprice：楼盘总价（万/套）
	
2.用户表 userlist
    id
    username: 用户名
    password：密码
    headimg：头像
3.标题表 titlelist
    id
    title: 标题
```
